Steps

1- git init 
2- git add .
3- git commit -m 'message of the commit'
4- git push

Branches
5-git branch (ver os Branches)
6- git branch Name of the Branche(Criar Branches)
7- git checkout banchName (Change branch)
8- git checkout -b branchName (Create a new branch a change it)

9-git merge branchName (To merge Branch content)
10- git pull ( to pull it to another branch )
