Steps

1- git init 
2- git add .
3-git branch -M branchName
4- git commit -m 'message of the commit'
5- git push

Branches
6-git branch (ver os Branches)
7- git branch Name of the Branche(Criar Branches)
8- git checkout banchName (Change branch)
9- git checkout -b branchName (Create a new branch a change it)

10-git merge branchName (To merge Branch content)
11- git pull ( to pull it to another branch )

git reset --mixed (elimina apenas o commit e volta os arquivos para a versão anterior )
git reset --hard ( ignora a resistencia do commit e tudo que foi feito nele)
